# Machine Learning Projects

This repository contains a collection of **machine learning projects** built using real-world datasets.  
The goal of this repository is to show the **core skills required for a Data Scientist** using practical examples.

---

## Skills Covered

### 1. Programming & Tools
- Python programming
- Pandas and NumPy for data handling
- Scikit-learn for machine learning
- Matplotlib and Seaborn for data visualization
- Jupyter Notebook
- Git and GitHub

---

### 2. Data Preparation
- Data cleaning and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Feature engineering
- Data scaling and transformation

---

### 3. Machine Learning Models
- Linear Regression, Ridge, Lasso
- Logistic Regression
- Decision Trees
- Random Forest
- Gradient Boosting / XGBoost
- K-Nearest Neighbors (KNN)
- Support Vector Machines (SVM)

---

### 4. Model Training & Tuning
- Train-test split
- Cross validation
- Hyperparameter tuning
- Preventing overfitting

---

### 5. Model Evaluation
- Regression metrics (RMSE, MAE, RÂ²)
- Classification metrics (Accuracy, Precision, Recall, F1-score, ROC-AUC)
- Model comparison and selection

---

### 6. Time Series Forecasting
- Trend and seasonality analysis
- ARIMA and SARIMA models
- Forecast evaluation
- Business use cases

---
## Project Structure

Each project includes:
- Problem statement
- Dataset description
- Modeling approach
- Results and insights

---

## Purpose
This repository is created to practice machine learning and to demonstrate the **end-to-end workflow of a Data Scientist**, from raw data to final insights.
